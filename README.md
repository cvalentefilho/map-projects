# map-project
My first react projet with google maps integration

### Used

```
[google-map-react](http://istarkov.github.io)
[react-maskedinput](https://github.com/insin/react-maskedinput)
```


## Run project local

```
git clone this repo
```


### npm

To install dependencies run:

```
npm install --save
```

To execute run:

```
npm run
```
